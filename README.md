# Roadly

Roadly — адаптивна автомобільна панель для телефона й планшета. Вона поєднує GPS-спідометр, ручний ліміт швидкості, швидкий запуск маршруту в Google Maps та керування демонстраційною чергою YouTube Music.

## Можливості

- фактична швидкість із браузерного Geolocation API;
- попередження про перевищення встановленого ліміту та вібросигнал;
- ручний ліміт 20–130 км/год із локальним збереженням;
- відстань і час поточної поїздки;
- маршрути через офіційний Google Maps Directions URL;
- швидке відкриття YouTube Music і демонстраційні медіаконтроли;
- голосове введення адреси там, де браузер підтримує Web Speech API;
- встановлення як PWA та базова офлайн-оболонка;
- адаптивний інтерфейс для телефона, планшета й широкого екрана.

## Запуск локально

```bash
python3 -m http.server 4173
```

Після цього відкрийте [http://localhost:4173](http://localhost:4173).

## Важливі обмеження

Roadly не є заміною штатної панелі автомобіля та не визначає дорожній ліміт автоматично. Для автоматичного ліміту потрібне окреме джерело дорожніх даних/API. Фактична швидкість залежить від якості GPS і дозволу браузера на геолокацію. YouTube Music і Google Maps відкриваються як зовнішні сервіси; повна інтеграція з акаунтом потребує їхніх офіційних SDK/API та авторизації.

Не взаємодійте з телефоном під час руху. Завжди керуйтеся дорожніми знаками та показниками автомобіля.

## Структура

- `index.html` — доступна семантична розмітка;
- `styles.css` — адаптивний автомобільний інтерфейс;
- `app.js` — GPS, спідометр, маршрут, музика й налаштування;
- `manifest.webmanifest`, `sw.js` — PWA та локальне кешування;
- `assets/icon.svg` — іконка застосунку.
